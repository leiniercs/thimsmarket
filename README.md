# [SDL Platforms - Pay](https://pay.sdlplatforms.com) website

## Building and running

Installing dependencies:

```sh
npm install
# or
yarn install
```

Building:

```sh
npm run build
# or
yarn build
```

Running (production mode):

```sh
npm run start
# or
yarn start
```

Open [http://localhost:3000](http://localhost:3000) with your browser.
